# Orbit OMS

Modern React-based Order Management Dashboard with a futuristic SaaS-inspired UI.

## Features

- Analytics dashboard
- Order tracking
- Responsive layout
- Modern dark UI
- Interactive charts
- Sidebar navigation
- Animated gradients and cards

## Tech Stack

- React
- Vite
- JavaScript
- CSS
- React Router
